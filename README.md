# Sentiment Analysis - NLP Project

A machine learning project that predicts whether a movie review is Positive or Negative.

## What it does
- Reads movie reviews as input
- Cleans and processes the text
- Predicts sentiment using Naive Bayes

## Results
- Dataset: 50,000 IMDB Movie Reviews
- Accuracy: 85.53%

## Tech Stack
- Python
- NLTK
- Scikit-learn
- Pandas

## How to run
1. Install libraries: pip install pandas nltk scikit-learn
2. Download IMDB Dataset.csv from Kaggle
3. Run: python main.py
